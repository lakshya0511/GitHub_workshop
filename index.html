Github and 101 Git

Hi , welcome to the workshop!

This workshop introduces the fundamentals of Git and GitHub, empowering developers to manage their code workflows efficiently, collaborate remotely, and contribute to great software projects.

We will cover the basics of Git version control, repositories, commits, branching, and how to leverage GitHub for team collaboration.

Workshop Goals

  • Understand Git fundamentals
  • Confidently use GitHub repositories
  • Master version control workflows
  • Collaborate effectively with others
